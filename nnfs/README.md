# Neural Networks from Scratch in Python

This repository contains my implementation of a Deep Learning Neural Network built entirely from scratch. 
## Overview
The primary goal of this project is to demystify the "black box" of deep learning. By programming the forward passes, calculating loss, and implementing backpropagation manually, this repository serves as a practical exploration of the mathematics that power modern neural networks.

##  Core Components Implemented
* **Dense Layers:** Custom implementation of hidden layers with adjustable neuron counts.
* **Activation Functions:** 
  * ReLU (Rectified Linear Unit) for hidden layers.
  * Softmax for the output layer.
* **Loss Functions:** Categorical Cross-Entropy Loss.
* **Optimization & Backpropagation:** Gradient calculations and weight/bias updates.
