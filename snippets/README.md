## Experiments & Mini-Projects

This directory contains a collection of miscellaneous scripts, mini-projects, and coding experiments related to Artificial Intelligence and Machine Learning. 

## About the Contents
The files in this folder are independent of each other. They are mainly used for:
* Testing new libraries and frameworks.
* Practicing specific algorithms.
* Storing useful code snippets.

The ideas, code snippets, and solutions here are compiled from various open-source platforms, anonymous blogs, community forums, and my personal trial-and-error processes. 

Since these are small-scale practices gathered from untracked or diverse sources across the internet, specific citations are not listed here. They are heavily modified and adapted for my personal learning journey.
