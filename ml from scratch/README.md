#  Machine Learning Algorithms From Scratch

This repository contains my personal Python implementations of fundamental Machine Learning algorithms. The codes here are written from scratch without relying on high-level ML libraries (like scikit-learn) to gain a deeper programmatic and mathematical understanding of how these models work under the hood.

## Overview
By building these algorithms from the ground up using plain Python and NumPy, this project explores the core mathematics, cost functions, and optimization techniques (such as Gradient Descent) that power modern AI systems. 

## Algorithms Implemented
* Linear Regression
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Support Vector Machines (SVM)
* Naive Bayes
* K-Means Clustering 
* Neural Network
